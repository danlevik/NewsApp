# Веб-приложение "Новостной сайт"
Данный проект представляет собой веб-приложение для просмотра и написания новостей

### Запуск
```docker compose up --build```

### Основные функции

|     Роль             	|     Функция                                  	|
|----------------------	|----------------------------------------------	|
|     Гость            	|     Просмотр   статей на главной странице    	|
|                      	|     Чтение статьи                            	|
|                      	|     Регистрация                              	|
|     Пользователь     	|     Авторизация                              	|
|                      	|     Редактирование   информации в профиле    	|
|                      	|     Комментирование   статей                 	|
|     Автор            	|     Создание   новых статей                  	|
|                      	|     Удаление   статей                        	|
|                      	|     Просмотр   своих статей в профиле        	|
|     Администратор    	|     Управление   пользователями              	|

### Стэк технологий
Backend: Java, фреймворк Spring
Frontend: HTML + CSS
